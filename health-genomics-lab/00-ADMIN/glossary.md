# Глоссарий

| Термин | Определение | Категория |
|--------|-------------|-----------|
| SNP | Single Nucleotide Polymorphism — замена одного нуклеотида в ДНК | Генетика |
| rs-номер | Референсный номер SNP (например rs1815739) | Генетика |
| Генотип | Комбинация аллелей (например AA, AG, GG) | Генетика |
| Аллель | Вариант гена | Генетика |
| HRV | Heart Rate Variability — вариабельность сердечного ритма | Физиология |
| MTHFR | Ген, отвечающий за фолатный цикл | Генетика |
| REM | Rapid Eye Movement — фаза быстрого сна | Сон |
| CRP | C-Reactive Protein — маркер воспаления | Биомаркер |
| HbA1c | Гликированный гемоглобин — показатель среднего уровня глюкозы | Биомаркер |
| VO2max | Максимальное потребление кислорода — показатель выносливости | Спорт |
